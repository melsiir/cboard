# cboard - Programming Keyboard for Android

cboard is a specialized Android keyboard designed for programming. It features extra rows for programming symbols, customizable layouts, and is optimized for efficient mobile coding.

## Features

- **Extra symbol rows**: Dedicated rows for common programming symbols like `{ } [ ] ( ) < > = ! & | * / - + % ^ $ # @`
- **Customizable layout**: Add, remove, or rearrange keys on any row
- **Adjustable settings**: Control keyboard height (30-70%) and button size (small/medium/large)
- **Export/Import settings**: Backup and restore your keyboard configuration
- **Material Design 3**: Beautiful, modern UI that follows Google's design guidelines
- **System theme support**: Automatically follows system light/dark theme and accent colors
- **Space-aware**: Optimized for comfortable mobile typing

## Installation

1. Clone or download this repository
2. Open the project in Android Studio
3. Build and install the APK on your device

Or download the pre-built APK from the GitHub Actions artifacts.

## Usage

1. After installation, go to Settings > System > Languages & Input > On-screen keyboard
2. Select cboard from the list
3. Enable cboard and set it as default
4. Open the cboard app to configure your keyboard preferences

## Building with GitHub Actions

This project includes a GitHub Actions workflow that automatically builds the APK on every push to the main branch. The APK can be downloaded from the workflow artifacts.

## Customization

The keyboard layout can be customized through the Settings > Customize Layout screen:
- Add new characters to any row
- Remove existing characters
- Reset to default layout
- Use common programming symbols quick-add buttons
